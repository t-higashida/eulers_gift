\section{数列の和}

\subsection{パスカルの三角形と数の和}

本節では，さらに考察を広げて，パスカルの三角形の第$n$段階までに現れるすべて
の数の和について考えよう．これは数列$S_n$の和
\[
	T	\equiv	S_0	+	S_1	+	S_2	+	S_3	+	S_4	+	\cdots	+	S_n
\]
を求めることである．上式を総和記号を用いて
\[
	T=	\sum_{l=0}^{n}	S_l
\]
と表し，\textbf{有限級数（finite series）}と呼ぶ．

初項が$1$，公比が$p$である等比数列の第$n$項までの和を
\[
	I	=	1	+	p	+	p^2	+	p^3	+	p^4	+	\cdots	+	p^n
\]
と書く．これは\textbf{等比級数}，あるいは\textbf{幾何級数}（geometirc series）と呼ばれる．

まず，この和を求めよう．上式の両辺に$p$をかけて
\[
	pI	=	p	+	p^2	+	p^3	+\cdots	+	p^{n+1}
\]
を作り，二式の辺々を引き算すると
\[
	(1-p)	I	=	1	-	p^{n+1}
\]
となる．$I$について解き，等比数列の総和を与える式
\[
	1	+	p	+	p^2	+	p^3	+	\cdots	+	p^n	=	\frac{1-p^{n+1}}{1-p}
\]
を得る
\footnote{
特に，$p=1$の場合は$I=n$である．
}これで全数の和$T$を求める準備が整った．数列$S_n$は公比が$2$なので，$p=2$を上式に代入して
\[
	T=	\frac{1-2^{n+1}}{1-2}	=	2^{n+1}	-1.
\]
よって，第$N$段までの数の総和は$2^{n+1}-1$と求められた
\footnote{
	これは面白いことに，先に述べたメルセンヌ数の形をしている．すなわち，
素数$2^{859433}-1$は$859432$段までのパスカルの三角形を内部の数をすべて加えたもの，とも考えられる．
}．
